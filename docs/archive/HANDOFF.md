# NL-DPE FPL Project Handoff

## Project Context
Research paper evaluating NL-DPE as a heterogeneous FPGA hard block for ML/DL acceleration, compared to Azure-Lily baseline.

## Completed Work

### 1. GEMV RTL + VTR Synthesis
- 5 GEMV shapes: `gemv_1_{K}_{N}` for (64,64), (512,128), (2048,256), (256,512), (512,512)
- RTL in `nl_dpe/gemv_*.v`, generated by `nl_dpe/gen_gemv_wrappers.py`
- VTR results validated: DPE counts match analytical tiling V×H

### 2. Attention Head RTL (log-domain DIMM)
- `nl_dpe/attention_head_1_channel.v` — complete RTL
- 3 DPEs (Q/K/V projections, ACAM=log mode)
- `dimm_score_matrix`: CLB add + exp + reduce (log-domain Q×K^T)
- `dimm_weighted_sum`: CLB log + add + exp + reduce
- `softmax_approx`: CLB exp + normalization
- VTR result: 3 DPEs, 89 CLBs, Fmax=36.84MHz, 18% device util on 550×550

### 3. IMC Simulator Integration
- `run_imc_with_vtr_freq.py`: patches IMC config with VTR Fmax, runs energy/latency analysis
- Energy params: e_analogue=3.89pJ, e_digital=0.171pJ/col, e_conv=0 (ACAM absorbs ADC)

### 4. DPE Area/Power Model (`nl_dpe/area_power.py`)
- `dpe_specs(row, col)` returns full dict: area, power, energy, VTR tile sizing
- ACAM dominates: 77% of DPE area, 91% of power, both scale linearly with cols
- VTR tile (W,H) computed via routing-aware formula:
  `DPE_logic + W×H×SB + (W+H)×CB ≈ W×H×CLB_tile`
  Constants: SB=688µm², CB=303µm², CLB_tile=2239µm²
- XML `area` tag = core logic area only (MWTA), consistent with CLB area=27905

## Critical Technical Details
- `dpe` is VTR black box: Verilog `.subckt dpe` → arch XML `pb_type name="nl_dpe"`
- Ports: data_in[39:0], data_out[39:0], nl_dpe_control[1:0], w_buf_en, shift_add_control/bypass, load_input/output_reg, clk, reset (40-bit bus)
- DPE tile: `<tile name="wc" height="{h}" width="{w}" area="{area_tag_mwta}">`
- DSP area: 253,779 MWTA (height=4, width=1). BRAM area: 137,668 MWTA (height=2, width=1)
- BRAM: height=2, startx=2, repeatx=16. Per replica: 4 BRAMs (K≤1024), 6 BRAMs (K>1024)
- DPE compute equivalent: X = R×C / 4 DSPs (9×9 SOP-4 mode, 4 int8 MACs/cycle)
- DPE storage equivalent: Y = R×C×8 / 36Kbit BRAMs
