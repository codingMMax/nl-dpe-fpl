=== Bugs Found and Fixed ===

Simulator:
  1. dimm_nonlinear() missing SRAM→DPE read stage (47% latency undercount)
  2. gemm_log() same missing read stage

RTL generator (gen_attention_wrapper.py):
  1. conv_controller write-address off-by-one → replaced with direct DPE
  2. Word-wise CLB adder → byte-wise (no carry between int8 lanes)
  3. 1-cycle SRAM latency assumption → 2-cycle (addr register + SRAM registered read)
  4. Flat K SRAM packing → padded per-key (word-aligned offsets)
  5. dimm_exp_valid timing: mac_count > 0 → mac_count >= 2
  6. ready_n polarity: 1'b0 (stall) → 1'b1 (ready)
  7. Missing DPE idle wait between scores → MSB_SA_Ready check
  8. Accumulator summed full 40-bit words → masked byte-wise sum (first d cols only)

Behavioral DPE (dpe_stub.v):
  1. Added ACAM_MODE parameter (0=none, 1=exp, 2=log)
  2. Weight memory zero-initialized (fixes X propagation)
